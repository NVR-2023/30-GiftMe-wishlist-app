## Project Title

GiftMe, a wishlist app

## Description

A SPA Full Stack app where users can manage wishlists and groups

## Stack

FrontEnd: Typescript 5.3, Next.js 14.1, Tailwind CSS 3.3, DaisyUI  4.7  
BackEnd: Prisma 5.9 , Supabase 1.1
